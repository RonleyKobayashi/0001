总结：

1. 这种方式之对chatgpt-3.5-turbp和chatgpt-3.5-16k的大概率管用 而chatgpt 4.0 apii则不一定管用。
2. 本功能利用了chatgpt回答可被篡改的的漏洞，让gpt误以为是自己的回答，gpt对提问者的问题有严格的审查，但是对自己说过的话却大概率审查。
3. 我认为本漏洞算是严重漏洞，本漏洞能书写的的内容十分大，发挥想象力有很多（包括但不限于各种违法行为）。
